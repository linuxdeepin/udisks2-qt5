# udisks2-qt5
UDisks2 DBus interfaces binding of Qt5. 

### Build dependencies

- qtbase5-dev
- pkg-config

## Installation

### Build from source code

``` shell
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ qmake ../
$ make
$ sudo make install
```
