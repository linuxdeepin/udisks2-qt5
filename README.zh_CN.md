# udisks2-qt5

使用 Qt5 封装的 UDisks2 的 DBus 接口。

### 编译依赖

- qtbase5-dev
- pkg-config

## 安装

### 构建过程

``` shell
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ qmake ../
$ make
$ sudo make install
```
